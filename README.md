# Tic Tac Toe Game (React)

## Description
This is a practice project I worked upon during my react learning days.
This project is a simple Tic Tac Toe game built using React.  
It demonstrates React concepts such as components, props, state management, and event handling.

The game allows two players to play alternately using X and O, detects the winner, and prevents overwriting already filled squares.

---

## Features

- Interactive Tic Tac Toe board
- X / O turn switching
- Winner detection
- Prevent clicking the same square twice
- Clean component structure
- Built using React with functional components

## How the Game Works

- The board contains 9 squares.
- Each click updates the state.
- Turns alternate between X and O.
- The game checks for winner after every move.
- If a square is already filled, it cannot be clicked again.
- Once a winner is found, no more moves are allowed.
